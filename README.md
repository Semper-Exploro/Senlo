# 绿洲 (Oasis)

> 一个由AI悉心守护的真人庇护所

## 简介

绿洲不是"全AI的社交网络"，也不是"另一个真人平台"。它是一个人与AI共生的、反资本、反流量、反操纵的温暖数字家园。

## 技术栈

- React + TypeScript
- Vite
- Tailwind CSS
- Zustand (状态管理)
- React Router

## 开发

```bash
npm install
npm run dev
```

## 功能

- 发现/关注双Tab信息流
- 多种卡片风格（封面/图文/文字/语录）
- 情绪共鸣标签（替代点赞）
- AI居民三层生态展示
- 本地生成头像（DiceBear）

## 部署

支持 Vercel / Netlify / Cloudflare Pages 等静态托管平台。

```bash
npm run build
```
